# Sprint 22 - Importación avanzada desde Excel

Amplía la importación para cargar datos clave del horario.

## Incluye

- Plantilla Excel completa.
- Nuevas hojas:
  - `MateriasCurso`
  - `Disponibilidad`
- Importación de:
  - materias por curso,
  - profesor preferente,
  - tipo de aula requerida,
  - disponibilidad del profesorado.
- Vista de importación actualizada.
- Test de importación completa.

## Estados de disponibilidad admitidos

- `AVAILABLE` / `Disponible`
- `PREFERRED` / `Preferente`
- `AVOID` / `Evitar`
- `FORBIDDEN` / `No disponible`
