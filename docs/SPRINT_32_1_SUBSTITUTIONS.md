# Sprint 32.1 - Motor de sustituciones inteligentes

Este sprint inicia el módulo de sustituciones.

## Incluye

- `Absence`
- `CandidateTeacher`
- `SubstitutionProposal`
- `SubstitutionAvailability`
- `ImpactAnalyzer`
- `SubstitutionRanker`
- `SubstitutionEngine`
- `SubstitutionService`
- Tests de disponibilidad, ranking y motor.

## Criterios iniciales

- Profesor libre.
- Profesor de guardia.
- Especialidad compatible.
- Mismo edificio.
- Apoyos que se pueden mover.
- Penalización por romper apoyos importantes.

## Siguiente paso

Conectar el motor con la base de datos real de Pini: profesorado, horarios, apoyos, guardias y disponibilidad.
