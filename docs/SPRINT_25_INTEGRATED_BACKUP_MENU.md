# Sprint 25 - Copias de seguridad integradas en la interfaz

Este sprint enlaza automáticamente el módulo de copias de seguridad en la ventana principal.

## Incluye

- Sustitución completa de `main_window.py`.
- Menú `Archivo > Copias de seguridad`.
- Botón `Copias de seguridad` en la pantalla de inicio.
- Import automático de `BackupView`.
- Sin parche manual.

## Requisito

Debe estar aplicado el Sprint 24, que añade:
- `BackupService`
- `BackupView`
