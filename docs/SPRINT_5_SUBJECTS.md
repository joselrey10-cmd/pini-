# Sprint 5 - Gestión de Materias

Este sprint añade la gestión de materias.

## Incluye

- Tabla SQLite `subjects`.
- Servicio `SubjectService`.
- Modelo `Subject`.
- Vista `SubjectsView`.
- Diálogo para crear y editar materias.
- Carga de materias base del CEIP Tierra de Pinares.
- Acceso desde menú `Datos > Materias`.
- Tests CRUD y carga inicial.

## Materias base

- Lengua.
- Matemáticas.
- Inglés.
- Educación Física.
- Música.
- Religión.
- Atención Educativa.
- Contextos.
