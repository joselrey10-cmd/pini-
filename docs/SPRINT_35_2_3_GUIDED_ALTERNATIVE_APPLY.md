# Sprint 35.2.3 - Aplicación guiada de alternativas

Añade una previsualización antes de aplicar alternativas inteligentes.

## Incluye

- `AlternativePreview`
- `AlternativePreviewService`
- `AlternativePreviewDialog`
- Botón `Previsualizar alternativa`
- Confirmación antes de aplicar alternativa
- Textos con:
  - destino,
  - mejora estimada,
  - score estimado,
  - explicación,
  - motivos.
- Tests

## Resultado

Pini ya no aplica alternativas a ciegas: primero muestra el impacto previsto y pide confirmación.
