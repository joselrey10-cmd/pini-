# Sprint 34.1 - Comparador de soluciones

Este sprint inicia la serie 34 de optimización avanzada.

## Incluye

- `SolutionComparator`
- `SolutionComparisonItem`
- `SolutionComparisonResult`
- `SolutionComparisonReport`
- Ranking de soluciones por puntuación.
- Identificación de la mejor solución.
- Tests.

## Utilidad

Pini puede comparar varias alternativas de horario y justificar cuál tiene mejor calidad.
