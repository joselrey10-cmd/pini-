# Sprint 3 - Gestión de Profesores

Este sprint añade la primera pantalla funcional de Pini.

## Incluye

- Tabla SQLite `teachers`.
- Servicio `TeacherService`.
- Modelo `Teacher`.
- Vista `TeachersView`.
- Diálogo para crear y editar profesores.
- Acceso desde menú `Datos > Profesores`.
- Test CRUD completo.

## Ejecutar

```powershell
$env:PYTHONPATH="apps/desktop;packages/core;packages/rule_engine;packages/scheduler;packages/persistence"
python -m pini_desktop
```
