# Sprint 31.3 - Búsqueda tabú

Este sprint añade un tercer algoritmo de optimización.

## Incluye

- `TabuSearchOptimizer`.
- `TabuSearchResult`.
- Lista tabú de soluciones recientes.
- Evita repetir movimientos o estados cercanos.
- Puede aceptar soluciones no mejores para explorar más.
- `SearchReport` ampliado con modo `tabu`.
- Tests del algoritmo.

## Importancia

La búsqueda tabú ayuda a evitar ciclos y mejora la exploración en problemas de horarios complejos.
