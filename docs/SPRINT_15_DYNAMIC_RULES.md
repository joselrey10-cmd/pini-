# Sprint 15 - Reglas dinámicas del centro

Añade un primer editor de condicionantes del centro.

## Incluye

- Tabla `dynamic_rules`.
- Servicio `DynamicRuleService`.
- Vista `DynamicRulesView`.
- Diálogo para crear/editar reglas.
- Reglas base del CEIP Tierra de Pinares:
  - máximo general de una sesión consecutiva,
  - Inglés 4º-6º permite dos consecutivas,
  - Contextos después del recreo.

## Siguiente paso
