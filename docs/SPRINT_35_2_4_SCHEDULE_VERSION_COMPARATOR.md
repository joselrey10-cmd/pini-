# Sprint 35.2.4 - Comparador de versiones de horario

Permite comparar dos versiones guardadas del horario.

## Incluye

- `ScheduleVersionComparator`
- `ScheduleVersionComparison`
- Detección de:
  - sesiones añadidas,
  - sesiones eliminadas,
  - sesiones movidas.
- `ScheduleVersionComparePanel`
- Tests.

## Siguiente paso

Integrar el panel de comparación dentro de la pestaña de versiones o crear una vista específica de comparación visual.
