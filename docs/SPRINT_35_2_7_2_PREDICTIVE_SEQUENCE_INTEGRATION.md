# Sprint 35.2.7.2 - Integración predictiva en cadenas IA

Integra la simulación predictiva dentro del panel de cadenas inteligentes.

## Incluye

- `PredictiveSequenceRanker`
- `PredictiveSequenceOptimizer`
- Botón `Simular impacto futuro`
- Visualización de:
  - score inmediato,
  - score predictivo,
  - delta final,
  - número de riesgos.
- Protección ante cadenas no recomendadas.
- Tests

## Resultado

Pini puede valorar si una cadena que mejora ahora seguirá siendo positiva después de considerar riesgos futuros.
