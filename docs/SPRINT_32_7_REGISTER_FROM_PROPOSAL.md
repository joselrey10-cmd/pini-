# Sprint 32.7 - Registrar sustitución desde la propuesta

Este sprint conecta la pantalla de propuestas con el registro de sustituciones.

## Incluye

- Botón `Guardar propuesta seleccionada`.
- Registro automático de:
  - profesor ausente,
  - sustituto,
  - día,
  - periodo,
  - puntuación,
  - motivos,
  - avisos.
- Integración de `Registro de sustituciones` en:
  - menú `Horarios`,
  - pantalla de inicio.
- Test de importación de vistas.

## Requisito

Debe estar aplicado el Sprint 32.6.
