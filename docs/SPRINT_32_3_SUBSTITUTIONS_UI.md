# Sprint 32.3 - Interfaz de sustituciones

Este sprint añade una vista gráfica para consultar sustituciones.

## Incluye

- `SubstitutionView`.
- Selector de profesor ausente.
- Día y periodo.
- Tabla de propuestas.
- Puntuación, motivos y avisos.
- Integración en `main_window.py`.
- Acceso desde:
  - `Horarios > Sustituciones`
  - pantalla de inicio.

## Utilidad

El director puede consultar sustituciones directamente desde Pini, usando el motor conectado a los datos reales.
