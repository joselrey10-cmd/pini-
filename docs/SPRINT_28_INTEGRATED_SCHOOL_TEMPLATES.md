# Sprint 28 - Plantillas automáticas integradas

Este sprint integra completamente la vista de plantillas automáticas en la interfaz principal.

## Incluye

- Sustitución completa de `main_window.py`.
- Menú `Archivo > Plantillas automáticas`.
- Menú `Centro > Plantillas automáticas`.
- Botón `Plantillas automáticas` en la pantalla de inicio.
- Import automático de `SchoolTemplateView`.
- Sin parche manual.

## Requisito

Debe estar aplicado el Sprint 27, que añade:
- `SchoolTemplateService`
- `SchoolTemplateView`
