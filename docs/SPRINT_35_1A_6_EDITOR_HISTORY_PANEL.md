# Sprint 35.1A.6 - Historial visual del editor

Añade un panel visual de historial a la matriz de horarios.

## Incluye

- `EditorHistoryPanel`
- Listado de acciones aplicadas
- Registro visual de:

  - intercambios,
  - deshacer,
  - rehacer,
  - selección/cancelación.
- Integración junto al panel de impacto.
- Test básico.
