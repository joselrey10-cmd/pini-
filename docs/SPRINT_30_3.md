# Sprint 30.3 - Primer optimizador funcional

Este sprint convierte el motor en un optimizador inicial.

## Incluye

- Intercambio simple de sesiones.
- Evaluación de conflictos:
  - profesor duplicado,
  - curso duplicado,
  - aula duplicada.
- Puntuación ponderada:
  - restricciones,
  - huecos de profesorado,
  - distribución de materias,
  - uso de aulas.
- Búsqueda local básica.
- Resultado de optimización con:
  - puntuación inicial,
  - puntuación final,
  - iteraciones,
  - mejora detectada.

## Importante

Este optimizador todavía es sencillo, pero ya compara soluciones y se queda con la mejor.
