# Sprint 33.7 - Historial de sincronizaciones

Registra cada sincronización o uso de caché offline.

## Incluye

- `SyncHistoryRecord`
- `SyncHistoryStore`
- Registro automático al sincronizar.
- Registro al usar caché offline.
- Listado y borrado de historial.
- Servicio desktop ampliado.
- Vista `EducaCyLHistoryView`.
- Archivo de parche para enlazar menú.
- Tests.

## Utilidad

Permite auditar qué se importó, cuándo, desde dónde y con qué cambios.
