# Sprint 7 - Materias por curso

Este sprint conecta cursos y materias.

## Incluye

- Tabla SQLite `course_subjects`.
- Servicio `CourseSubjectService`.
- Modelo `CourseSubject`.
- Vista `CourseSubjectsView`.
- Diálogo para asignar materia a curso.
- Selección de profesor preferente.
- Sesiones semanales específicas por curso.
- Tipo de aula requerido.
- Test CRUD completo.

## Importancia

Este módulo es clave para el futuro motor de horarios: define cuántas sesiones debe impartir cada materia en cada curso y qué profesor/a es preferente.
