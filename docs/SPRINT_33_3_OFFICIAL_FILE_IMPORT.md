# Sprint 33.3 - Importación desde archivos oficiales

Añade soporte para importar datos desde archivos locales exportados oficialmente.

## Incluye

- `OfficialFileParser`
- `OfficialImportTemplate`
- Soporte Excel:
  - Profesores
  - Cursos
  - Materias
  - Aulas
  - Centro
- Soporte CSV por columna `tipo`
- `sync_from_file`
- Servicio desktop ampliado
- Tests

## Importante

No se conecta a EducaCyL real. Trabaja con archivos oficiales/exportados/autorizados.
