# Sprint 31.4 - Algoritmo genético

Este sprint añade un cuarto algoritmo de optimización.

## Incluye

- `GeneticOptimizer`.
- `GeneticResult`.
- Población inicial.
- Selección por torneo.
- Cruce de soluciones.
- Mutación por intercambio de sesiones.
- Elitismo.
- `SearchReport` ampliado con modo `genetic`.
- Tests del algoritmo.

## Importancia

Pini empieza a explorar muchas soluciones simultáneamente, acercándose a estrategias usadas en problemas de optimización complejos.
