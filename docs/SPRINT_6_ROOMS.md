# Sprint 6 - Gestión de Aulas

Este sprint añade la gestión de aulas y espacios.

## Incluye

- Tabla SQLite `rooms`.
- Servicio `RoomService`.
- Modelo `Room`.
- Vista `RoomsView`.
- Diálogo para crear y editar aulas.
- Carga de aulas base.
- Acceso desde menú `Datos > Aulas`.
- Tests CRUD y carga inicial.

## Aulas base

- Aulas ordinarias.
- Gimnasio.
- Aula de Música.
- PT.
- AL.
- Biblioteca.
- Aula TIC.
