# Sprint 32.5 - Interfaz para ausencias de varios periodos

Este sprint actualiza la vista de sustituciones para gestionar tramos horarios.

## Incluye

- Selector de profesor ausente.
- Día.
- Periodo inicial.
- Periodo final.
- Consulta de sustituciones por cada periodo.
- Tabla con:
  - periodo,
  - profesor propuesto,
  - puntuación,
  - motivos,
  - avisos.
- Detección de tramos incorrectos.
- Visualización de periodos sin propuesta.

## Requisito

Debe estar aplicado el Sprint 32.4.
