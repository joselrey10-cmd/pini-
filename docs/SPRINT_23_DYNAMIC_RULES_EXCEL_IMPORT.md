# Sprint 23 - Importación de reglas dinámicas desde Excel

Amplía la plantilla e importación para incluir reglas dinámicas del centro.

## Incluye

- Nueva hoja `Reglas`.
- Importación de:
  - código,
  - nombre,
  - ámbito,
  - tipo,
  - prioridad,
  - afectado,
  - día,
  - periodo,
  - valor,
  - activa,
  - notas.
- Vista de importación actualizada.
- Test específico de importación de reglas.

## Utilidad

Permite añadir condicionantes nuevos durante el curso sin introducirlos uno a uno.
