# Sprint 32.8 - Informes de sustituciones

Este sprint añade informes básicos del registro de sustituciones.

## Incluye

- `SubstitutionReportService`.
- Resumen:
  - total,
  - planificadas,
  - realizadas,
  - canceladas.
- Exportación Excel del registro.
- Vista `SubstitutionReportView`.
- Test de resumen y exportación.
