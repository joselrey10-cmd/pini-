# Sprint 35.2.1.5 - AlternativeGenerator v1 integrado

Integra el generador de alternativas inteligentes en la matriz de horarios.

## Incluye

- `AlternativeApplyService`
- `AlternativeApplyResult`
- `AlternativesPanel` con botón de aplicar
- Señal `alternativeApplied`
- Nueva pestaña lateral `Asistente IA`
- Botón `Generar alternativas`
- Menú contextual:
  - Generar alternativas para esta sesión
- Integración con:
  - impacto,
  - historial,
  - optimización local,
  - recarga de matriz.

## Resultado

Pini ya puede generar alternativas para una sesión seleccionada y aplicar una propuesta con un clic.
